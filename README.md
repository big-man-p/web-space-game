# Web Space Shooter
**A lightweight space shooter game and mini game engine built from scratch using vanilla HTML, CSS, and JavaScript.**

This project was created to explore how core game engine concepts can be implemented in the browser without using any external libraries or frameworks. All rendering, physics, collision detection, and game logic are handled manually using the native web platform.
